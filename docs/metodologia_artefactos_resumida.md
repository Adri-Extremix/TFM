# Identificación de Artefactos - Guía Resumida

## Proceso de 2 Pasos

### Paso 1: Value Proposition Canvas
- Identificar stakeholder principal
- Jobs: tareas que necesita hacer
- Pains: frustraciones actuales
- Gains: beneficios esperados
- Artefacto que resuelve pain y crea gain

### Paso 2: Validación con Innovación
- Aplicar SCAMPER para mejoras
- Verificar viabilidad
- Asegurar que es concreto y accionable

## Reglas de Output
- Nombre claro del artefacto (máx 100 caracteres)
- Debe ser producto/servicio/sistema específico
- Relación directa con GIA y objetivo negocio
