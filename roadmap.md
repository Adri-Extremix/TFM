# RoadMap TFM
1. Tener clara la pipeline de procesos
    1. Identificar los procesos de toma de decisiones
    2. Identificar las entradas y salidas de cada proceso
    3. Definir las conexiones entre procesos
    4. Identificar dependencias entre procesos y datos
2. Definir claramente los datos de entrada y salida de cada proceso
    1. Documentar los datos de entrada en formato JSON
    2. Documentar los datos de salida en formato JSON
3. Transformar los procesos de formato Excel a un script de python
    1. SIPAC a python
    2. AHP a python
4. Definir tareas de los agentes para cada proceso
    1. Asignar responsabilidades a cada agente
    2. Definir las interacciones entre agentes
